método.filter()

El método .filter(). crea un nuevo array con todos los elementos que cumplan la condición implementada por la función dada.

Puede usarse en:
  • arrays String
  • arrays de Números

Sintaxis

 function filtrarNumeros(value) {
     const over18 = numbers.filter((number)=>number>18);
     document.getElementById("demo").innerHTML = over18;
 }

Ejemplo con string:

const miArray = ['zanahoria', 'queso', 'pan', 'jamón', 'tómate', 'cebolla'];
const resultado = miArray.filter((palabra) => palabra.length > 6);

resultado = ['Zanahoria','Tómate', 'Cebolla' ]
