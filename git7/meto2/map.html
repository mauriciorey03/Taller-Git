METODO Array.prototype.map()

El método Array.map() te permite iterar sobre un arreglo y modificar sus elementos utilizando una función callback. La función callback se ejecutará entonces en cada uno de los elementos del arreglo.

SINTAXIS

Parametros

Tienes este arreglo

let arr = [3, 4, 5, 6];

Ahora imagina que tienes que multiplicar cada uno de los elementos del arreglo por 3

let arr = [3, 4, 5, 6];

for (let i = 0; i < arr.length; i++){
arr[i] = arr[i] * 3;
}

console.log(arr); // [9, 12, 15, 18]

puedes utilizar el método Array.map() para conseguir el mismo resultado.

let arr = [3, 4, 5, 6];

let modifiedArr = arr.map(function(element){
return element *3;
});

console.log(modifiedArr); // [9, 12, 15, 18]
